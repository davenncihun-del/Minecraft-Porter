What this porter does

  • Analyzes archive contents including Java sources, metadata, assets, data packs, and config files.
  • Applies compatibility migration rules for loader API changes and common version transitions.
  • Writes a new archive named with the updated suffix instead of overwriting the original.
  • Produces a compatibility report showing applied rules, changed files, and unresolved issues.
